# M24AID008_CSL7110-Minhash-LSH-Assignment
Implementation of MinHash and Locality Sensitive Hashing for document similarity detection.

## Methods Implemented

* Shingling (2-character, 3-character, and 2-word)
* Jaccard similarity computation
* MinHash signature generation
* Locality Sensitive Hashing (LSH)
* Similar document candidate detection

## Files

* `CSL7110_minhash_assignment.ipynb` – Google Colab notebook implementing the complete pipeline.

## Tools Used

* Python
* Google Colab
* MinHash algorithm
* Locality Sensitive Hashing
